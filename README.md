# grok-voxel-heart-club
A voxel-style wild west town simulation web game featuring Heart (Ruby) and Club (Jax) outlaws. Drag characters around the blocky town square, trigger random events (tumbleweeds, weather, card games, money), chat in-character, and watch the duo interact. Pure HTML/CSS/JS with Tailwind. Future upgrades: full Canvas voxel rendering &amp; Three.js.
